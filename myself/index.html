计划（规划）

该栏目新建于 2024 年，主要用于记录该站博主的计划和完成情况，放在博客上一方面可以起到监督作用（自己装的逼，死也要装完），另一方面也可以增加博客更新频率，让搜索引擎误以为博主非常勤快。

该栏目将保持大概一月一更，每次更新都会新增加部分进展。虽然定下的计划可能不是很多，但我会争取每一个都完成。

2024 年上半年（完）

该栏目开始于 2024 年 3 月份。

  • 保持运动，争取每月跑 10-20 公里，并在年中做到 3 公里配速 5 分 20 秒。

其实去年就有这个想法，并且坚持了几个月。结果出于各种各样的原因在去年年底又没坚持住。

今年的各种事情让我意识到了我的身体确实还需要更强健些才行，故有此计划。

当前已完成三公里配速 5 分 20 秒。

  • 读一本尼采的书。

为什么是尼采？这件事说来话长。

其实以前的我对于抽象的哲学是嗤之以鼻的，直到前段时间在 B 站看到一个视频介绍尼采的思想和作品。

这个视频给我的触动很大，我决定亲自领略尼采的风骚。

目前进度：已阅读完《查拉图斯特拉如是说》。

感想：尼采真它宝贝的是个疯子，我应该是不会再看下一本了。

  • 学习 Unity & C#，并成功独立完成一个小项目。

一直有多学习几门代码的想法，但其实现在我除了 Python 会些，R 会点皮毛以外，也不会什么编程语言。

之前尝试学了一会 C++，然后就被 C++ 无比深奥的内存管理给劝退了（特指指针！）。

C# 有自动内存管理功能，整体学起来像是 Python 和 C++ 的折中，让我感觉还算可以接受，故有此计划。

更新：目前 C# 和 Unity 的基础知识已经学习完，但并没有实现一个完整的项目，该规划被博主判定失败。

  • 在 Kaggle 的一个 Competition 中，最终成绩达到前 5%。

虽然技多不压身，但技贵在精而不在多，所以我并不打算懈怠 Python 方面的学习。

这其中最感兴趣的一块就是 Machine Learning 了，看大佬们在炼丹时奇招频出真的令人神往。

所以这学期争取能在 Competition 里面搞出一点成绩。

目前进度：

  • 最近的一次 PlayGround 中拿到 top 1% (22/2788)。尝试进军前十中。
Don't blindly believe PB is all you need...
  • 最近的一次 Featured 中取得银牌（184/3856）。

  • 继续运动，争取将体重控制回 67-69kg。
