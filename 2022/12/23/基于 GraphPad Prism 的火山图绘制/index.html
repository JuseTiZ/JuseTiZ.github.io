基于 GraphPad Prism 的火山图绘制

发表于2022-12-23|更新于2022-12-23|学习绘图
|字数总计:910|阅读时长:2分钟|阅读量:

前排提示

此文中使用的 GraphPad Prism 版本为 9.4.1，获取方式请自寻，请支持正版。

需要准备的文件：差异表达分析结果（DESeq2、edgeR 或 limma 皆可）。

9.0 以上版本的 GraphPad Prism 貌似已经支持火山图绘制（自带），但是绘图效果并没有利用散点图绘制来的好，因此这篇文还是通过散点图的方式来实现火山图绘制，如果想要试试 Prism 自带的火山图可以在点开 Prism 时选择 Graph Portfolio 里的 Volcano Plot 进行：

绘图过程

在进行绘图之前，我们需要先筛选出 |log2FC| > 1 以及 adjustp < 0.05 的点，然后将它们的 log2FC 放在新的一列，最后效果应该如下所示：

具体的实现过程可以通过 Python 脚本或者 R 进行，在处理以后的表格就可以用来绘制火山图了。

点进 GraphPad Prism 后，选择最上方的 XY，各项选择如图：

生成表格后，导入差异表达分析结果（直接复制黏贴整个表格即可），此后删掉除了所需要的数据以外的列，最后如下图：

其中第一列为基因的 ID，第二列即 X 为各个基因的 log2FoldChange 值，后面三列 Y 为 adjust.p 值。此后选中三列 p 值，并点击箭头所指地方的 Analyze 选项：

选择 Transform 并点击 OK，此后选择转变 Y 值，选择 y = -1*Log(Y) ：

上述步骤如果嫌麻烦可以在 R 或 Python 的处理中顺便完成，在这里就展示下如何用 Prism 进行数据转换。

此后就能看见生成了一个新的表格和对应的图，点进图中可以看到它大致是长这个样子的：

到这里，一个初步的火山图雏形就已经产生，点击上方的 Format axes 进行进一步的润色：

在这里我就直接展示我在各个部分修改的参数：

y 轴设范围 -0.2 是因为避免等一下修改点大小后点挤出框外，更改完后整体效果应该如下：

此后我们在对点进行一些润色，同样直接以图的形式展现步骤：

点的大小可以选 2 也可以选 3，看个人的喜好而定。

最后，再修改下标签的名称和各标题，一个看上去还挺正经的火山图就诞生了：

到这里使用 GraphPad Prism 绘制火山图的分享就结束了，更进阶一些的使用就是基于这些功能再加一些操作而已。

例如可以用 Prism 展示各类基因的差异表达情况：

题外话

没想到最后真的出了这一篇类似于教程的分享，总体来说写的还挺顺畅的就是图有点多。

美中不足的地方是为了省一些加载时间选了一个相当小的数据集，导致没啥差异表达基因所以呈现的效果也差了些。

事实上 Prism 的火山图还能有更多的进阶用途，比如说刚才在最后展示的用于观察不同类基因差异表达情况的火山图，因为 Prism 自带了很好看的颜色分配机制和各数据集散点的图层分配管理等，所以能够支持很多具体想法的实现。

而且相关的教程在网上也能见到挺多，所以说这篇文的最主要价值可能是介绍 GraphPad Prism 这个画图软件。
